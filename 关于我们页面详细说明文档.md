# 关于我们页面详细说明文档

## 1. 概述

关于我们页面（AboutPage.vue）是生物科技企业官网的重要页面之一，主要展示企业形象、发展历程、核心优势和联系方式等信息。该页面采用现代化设计理念，通过丰富的视觉元素和交互体验，为用户呈现专业、可信的企业形象。

## 2. 技术架构

### 2.1 技术栈
- **前端框架**: Vue 3 + TypeScript
- **UI组件**: 自定义组件 + Tailwind CSS
- **状态管理**: Pinia
- **路由管理**: Vue Router
- **图片处理**: 响应式图片加载 + 错误回退

### 2.2 项目结构
```
src/views/about/
└── AboutPage.vue                    # 关于我们主页面

src/components/common/
├── ShowcaseBanner.vue              # 展示横幅组件
└── AppHeader.vue                   # 顶部导航栏

src/assets/styles/main.css          # 全局样式
src/stores/pageContentStore.ts      # 页面内容状态管理
src/types/view-data.ts              # 类型定义文件
public/images/common/
└── wx-qrcode-contact.png           # 微信客服二维码
```

### 4.3 响应式设计
页面采用移动优先的响应式设计：

**断点设置：**
- **移动端**: < 640px
- **平板端**: 640px - 1024px
- **桌面端**: 1024px+

**适配策略：**
- 横幅区域：调整标语字体大小和间距
- 统计数据：调整网格布局间距
- 联系方式：在小屏幕上改为垂直布局
- 时间线：调整内容宽度为100%

## 6. 性能优化策略

### 6.1 图片优化
- **二维码图片**: 适当大小的PNG格式，支持快速加载
- **图标使用**: Font Awesome字体图标，支持缩放和颜色变化
- **懒加载**: 大型图片支持懒加载机制

### 6.2 CSS优化
- **Tailwind CSS**: 按需生成，减少文件大小
- **关键CSS**: 内联关键路径CSS，提升首屏渲染速度
- **动画优化**: 使用transform和opacity，避免触发布局重绘

### 6.3 代码分割
- **路由懒加载**: AboutPage组件采用动态导入
- **组件按需加载**: ShowcaseBanner组件独立加载
- **样式隔离**: 使用scoped CSS避免样式污染

## 7. 用户体验设计

### 7.1 视觉层次
- **标题层级**: 清晰的H1-H3标题结构
