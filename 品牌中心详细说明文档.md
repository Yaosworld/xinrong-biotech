# 品牌中心页面详细说明文档

## 一、页面概述

品牌中心是网站的的核心板块之一，专注于展示公司旗下自主品牌和代理品牌的详细信息。该页面采用现代化设计，结合动态效果和响应式布局，为用户提供沉浸式的品牌浏览体验。

### 1.1 页面定位与功能

- **品牌展示**：集中展示所有自主品牌和代理品牌
- **品牌分类**：支持按品牌类型（自主品牌/代理品牌）和地域（国内/国际）分类
- **品牌详情**：提供每个品牌的详细介绍、Logo展示和代理授权证书
- **视觉体验**：采用粒子动画和几何装饰效果，提升页面视觉吸引力

### 1.2 技术栈

- **前端框架**：Vue 3 + TypeScript
- **状态管理**：Pinia
- **路由管理**：Vue Router
- **样式方案**：Tailwind CSS + SCSS
- **数据格式**：JSON
- **构建工具**：Vite

## 二、页面结构设计

### 2.1 信息架构

```

├── 展示横幅区
```

- **色彩方案**：以蓝紫渐变色为主色调（#667eea → #764ba2）

- **响应式设计**：完美适配桌面端和移动端
